Front-end fully responsive page
Built with Webpack, sass, js, html
Plugins used: tiny-slider

To run the app locally:
1. npm install
2. npm run build
3. npm run start

Available commands:
1. npm run start - starting local webpack server
2. npm run build - compile html, js and images to public folder
3. npm run watch - running build command on code change



The URL provided for fetching posts data was blocked by CORS policy
and with "no-cors" option returns empty body,
so I used "https://randomuser.me/api/?results=10" free API 
just to show implementation of fetch request as an example.
